# Zero Byte

### A Zero Byte é o nosso projeto de uma loja virtual de tecnologia, focada em peças de computador


## Como foi feito

### As tecnologias usadas neste projeto foram: HTML, CSS, Angular (TypeScript) e Spring Boot (Java)
